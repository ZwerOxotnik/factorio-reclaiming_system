Reclaiming system
-----------------------

Use /reclaim to claim entities from hostile teams as yours.\
This mod is designed for PvP, TvT and some PvE scenarios.

<p align="center">
  <a href="https://mods.factorio.com/mod/reclaiming_system/downloads"><strong>Download the mod&nbsp;&nbsp;‚ñ∂</strong></a>
</p>

Requirements
------------

Shell [scripts](./.scripts) depends on [git], [7z], [jq].

Installation on Debian and Ubuntu:

```shell
sudo apt install p7zip-full jq git -y
```
